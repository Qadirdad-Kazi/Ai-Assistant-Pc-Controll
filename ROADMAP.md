# Wolf AI - Future Implementation Roadmap

This document tracks the implementation of new features for the Wolf AI assistant.

## Phase 1: Knowledge & Memory
- [ ] **Grimoire (Local RAG / Knowledge Base)**
  - **Concept:** A dedicated tab for personal documents (PDFs, notes, code).
  - **Tech:** LangChain, ChromaDB (or FAISS), Sentence Transformers.
  - **Integration:** Allows Ollama to reference these docs during chat.
  - **Status:** *Pending*

## Phase 2: Entertainment & Media
- [ ] **Sonic Interface (Media Controller)**
  - **Concept:** Unified media player with "Wolf Knight" aesthetics.
  - **Tech:** Spotify API / System Media Transport Controls.
  - **Visuals:** Audio frequency visualizers.
  - **Status:** *Pending*

## Phase 3: System Intelligence
- [ ] **Sentinel (Advanced System Telemetry)**
  - **Concept:** Expansion of System Monitor into a full hardware dashboard.
  - **Tech:** `psutil`, `pyqtgraph` (or `matplotlib` embedded).
  - **Features:** Thermal graphs, network history, fan speed alerts.
  - **Status:** *Pending*

## Phase 4: Automation
- [ ] **Digital Janitor (File Automation)**
  - **Concept:** Background agent to organize file systems.
  - **Tech:** `watchdog` library.
  - **Status:** *Pending*

## Phase 5: Engagement
- [ ] **Gamification System**
  - **Concept:** XP and leveling system based on user productivity.
  - **Visuals:** XP Bar, unlockable themes/avatars.
  - **Status:** *Pending*

## Phase 6: Connectivity
- [ ] **Pack Link (Local Network Messaging)**
  - **Concept:** LAN communication between Wolf AI instances.
  - **Features:** Clipboard sharing, remote commands.
  - **Status:** *Pending*
