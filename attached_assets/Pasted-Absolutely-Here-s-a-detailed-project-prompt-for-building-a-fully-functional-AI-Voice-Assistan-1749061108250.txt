# 🧠 Project Prompt: **"Wolf: Your Smart PC AI Voice Assistant"**

**Goal**:
Build a cross-platform AI voice assistant named **Wolf** that wakes up on voice trigger (e.g., “Hey Wolf”), understands commands in English **and Urdu**, and can perform system-level tasks (like opening/closing apps, shutting down the system, reading emails, etc.) using **natural language** via **Ollama (LLM)**.

---

## ✅ Core Features

### 🎤 **Wake Word Activation**

* Triggers when hearing:

  * "Hey Wolf"
  * "Hi Wolf"
  * "Hello Wolf"
* Optional: Always-on background listener using VAD (Voice Activity Detection)

### 💡 **Multi-language Understanding**

* Detect and switch to **Urdu** via commands:

  * "کیا آپ اردو میں بات کر سکتے ہو؟"
  * “Can you reply in Urdu?”
* Responds in user’s preferred language (via language flag)

### 🖥️ **PC Control Commands**

* Close the current window
* Open apps (Chrome, Notepad, VSCode, etc.)
* Minimize/Maximize/Close window
* Adjust system volume
* Control media (play, pause, skip)
* Take screenshots
* Shut down, restart, log off

### 🧠 **AI Chat Functionality (via Ollama)**

* Conversational chatbot in both English and Urdu
* Write emails, summaries, notes, reminders
* Translate sentences
* Generate jokes, stories, ideas

### 📧 **Productivity Tools**

* Dictate and compose emails
* Summarize clipboard content
* Remind me to \[task] at \[time]
* Set timers and alarms

### 👨‍💼👩‍💼 **Voice Personalization**

* User can toggle between **Male** and **Female** voice
* Uses TTS libraries like:

  * `pyttsx3` or Google Cloud TTS for voice selection
  * Supports Urdu speech output via Google TTS

### 📡 **Live Notifications**

* Weather updates
* System battery status
* Meeting reminders
* Incoming emails or messages

---

## 🎨 UI Features

### 📋 Main UI

* Sleek, dark/light toggle
* Animated waveform when listening
* User chat + assistant chat in bubbles (like a chatbot)
* Voice settings: choose gender, accent, language
* Quick command buttons (Shutdown, Notes, Summary, Translate)
* Urdu/English toggle

### 📈 Analytics Page

* Logs of past voice interactions
* Most used commands
* Language usage pie chart

---

## 🔧 Tech Stack

### 🎛 Frontend (UI)

* **Electron.js** + **React.js** OR PyQT5 for native-like desktop UI
* TailwindCSS or CSS3 Animations for visuals

### 🧠 Backend

* **Python** (core logic)
* **Ollama LLM** via `ollama run llama2` or similar model
* `SpeechRecognition` / `Vosk` or `whisper.cpp` for transcription
* `pyttsx3` or Google Cloud TTS for TTS
* `pynput`, `pyautogui`, `subprocess` for PC control

### 🌍 Language Translation

* `googletrans` or **OpenAI** multilingual prompts
